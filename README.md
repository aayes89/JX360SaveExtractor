# JX360SaveExtractor
Extract data from save files of Xbox 360.

# Data Extracted (Examples)
<h3>Metadata</h3>

* File Type: CON, LIVE, PIRS
* Title ID: Hex value
* Profile ID: Hex value
* Console ID: Hex value
* Device ID: Hex value
* Title Name: Title of game data

<h3>Embedded Files</h3>

* Images (PNG, GIF, JPEG)
* Source code (if exists)
* Random data (DTA)
